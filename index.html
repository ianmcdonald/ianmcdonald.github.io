Info

Ian McDonald is a designer, developer, and educator from St. Catharines, Canada. He teaches graphic design at Niagara College and creates objects and artifacts for the Web. Previously at Form & Affect. Currently studying arts and culture at Brock University.

This website is hand-coded in Emacs, hosted on GitHub Pages, and uses system default fonts. It was last modified on February 2.
